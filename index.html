⚙️ Configuração do Banco de Dados (Supabase)

Configure uma vez. Após salvar, todos os dispositivos que acessarem este link compartilharão os mesmos dados em tempo real.

Passo 1: Acesse supabase.com → clique em "Start your project" → crie conta gratuita com Google ou e-mail.
Passo 2: Clique em "New project" → dê o nome daf-trindade → escolha uma senha → região South America (São Paulo) → clique "Create new project". Aguarde ~2 minutos.
Passo 3: No menu lateral, clique em SQL Editor → cole o código SQL abaixo e clique em Run:
CREATE TABLE IF NOT EXISTS solicitacoes (
  id bigint generated always as identity primary key,
  protocolo text not null,
  data_cadastro text,
  criado_em timestamptz default now(),
  situacao text,
  tipo text,
  nome text not null,
  data_nasc date,
  idade text,
  sexo text,
  sus text,
  mae text,
  endereco text,
  setor text,
  tel1 text,
  tel2 text,
  descricao text,
  servidor text,
  obs text,
  verificado boolean default false,
  verificado_em text
);

ALTER TABLE solicitacoes ENABLE ROW LEVEL SECURITY;
CREATE POLICY "allow_all" ON solicitacoes FOR ALL USING (true) WITH CHECK (true);
Passo 4: Vá em Project Settings → API (ícone de engrenagem no menu). Copie a Project URL e a chave anon public (em "Project API keys").

⚠️ Banco de dados não configurado

Para que os cadastros sejam compartilhados entre todos os dispositivos, configure o banco de dados Supabase na aba ⚙️ Configurar.
